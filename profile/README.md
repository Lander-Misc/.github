# Miscallaneous

Projects and experiments that don't fit cleanly into any other category.

## [i10](https://github.com/Lander-Misc/i10)

The first incarnation of my tiling window manager experimentation, written in AutoHotKey for Windows 10.

## [carts](https://github.com/Lander-Misc/carts)

A selection of Pico-8 libraries and experiments, grouped into one repo for the sake of organization.

Notably challenging platform due to limited code space.

## [TrenchBroom](https://github.com/Lander-Misc/TrenchBroom)

Fork of the TrenchBroom map editor for Quake, created to add basic dark mode support via Qt stylesheets.

## [Folded1000Times](https://github.com/Lander-Misc/Folded1000Times)

Mod-of-a-mod for Doom, implements Devil May Cry -style air juggle mechanics atop an existing melee combat system.
